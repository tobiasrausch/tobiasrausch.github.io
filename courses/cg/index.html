Analytical Methods in Cancer and Population Genomics

Course Content

This part of the course will focus on the analysis of short-read and long-read sequencing data from population and cancer genomics studies. Bioinformatic concepts, tools and methods required to analyse (tumor) sequencing data will be introduced. Learning outcomes include an overview of the challenges in the study of (cancer) genomic data sets, discovery and visualisation of copy-number and structural variants, understanding the principles of tumor purity, heterogeneity and ploidy and an overview of cancer epigenetics. The course covers different sequencing data modalities (short-reads and long-reads) and data types (bulk and single-cell). Practical data analysis sessions will complement the course.

Exercise 1: Variant Calling (due date 9th December 2025)

The goal of this exercise is to create a simple variant calling workflow for human genomic sequencing data. The workflow should contain steps to align the FASTQ files to the human reference genome (bwa), call variants (bcftools) and annotate all single-nucleotide variants with (VEP). For this excercise you can ignore all insertions and deletions, only single-nucleotide variants (SNVs) should be considered. The data has been subsampled to chr7 to speed-up all analysis. Once you have finished the exercise, please submit your answer using this Google Form.
  • Chromosome 7 human reference, chr7
  • FASTQ of read1, Read1
  • FASTQ of read2, Read2

Exercise 2: Cancer Genomics Data Analysis (due date 23rd December 2025)

In this exercise we want to analyze a cancer genomics sample, namely a paired tumor-normal sample pair. You can download the FASTQ data for tumor (tu) and matched control (wt) as well as the chrX reference FASTA file here. The main objective of this exercise is to align the data to the provided reference genome, to sort and index the alignments and to generate a read-depth plot similar to the one presented in the lecture. Please note that I downsampled the dataset and I also just kept the data for a 20Mbp region of chromosome X because otherwise all analysis take a lot of time for a full human genome. Once you have generated the alignment in BAM format you can count reads in 10Kbp windows using one of the tools shown below and compute the log2 read-depth ratio for each window of the tumor genome against the control. You can then plot the log2 read-depth ratio values using R or python, for example. Once you have finished the exercise, please submit your answer using this Google Form.
