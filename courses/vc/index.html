# Alignment and variant calling practical using FreeBayes and Delly

A tutorial on variant calling for next-generation sequencing data. This tutorial is written in markdown language and the content is tracked in the Variant Calling GitHub repository, please open an issue there if you notice any errors.

For questions, help or any feedback on the tutorial you are welcome to send me a message.
