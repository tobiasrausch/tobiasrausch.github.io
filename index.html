DELLY

  • DELLY is an integrated split-read and paired-end structural variant discovery and genotyping method suited for massively parallel sequencing data.

Other Software

  • nRex is a meta-method for somatic single-nucleotide variant calling.
  • I previously worked on SeqAn an open source library for sequence analysis algorithms, implementing graph algorithms and algorithms for multiple sequence alignment.
  • I wrote various quality control tools for short read illumina data which I recently integrated into Delly.
